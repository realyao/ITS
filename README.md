# Smart-traffic

### 基于树莓派的智慧交通模拟沙盘

* Base: `RaspberryPie`, `RFID`, `Sensor`,`Camera`...
* Function: `Electronic Toll Collection sys(ETC)`, `Vehicle obstacle avoidance`, `Automatic cruise` ...

### 部分展示
#### 1. 自动驾驶小车
  
  基于树莓派控制，实现自动循迹行驶，自动避障，远程操控等功能
